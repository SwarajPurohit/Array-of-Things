1. There are a total of 4 files(excluding README) and 2 folders.
2. The 'SO2 modelling.R' file is not required for the app. It shows how the forecasting model was developed.
3. The 'shiny_final.R' is the main file which needs to be run.
4. To install tensorflow in R studio, run the installation code given at the start of the script. Further details can be found in the link
   given in report references.
5. Once all the 4 files and 2 folders are in the working directory, the app can be launched by running the whole 'shiny_final.R' script.
